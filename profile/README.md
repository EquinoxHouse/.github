# Equinox House
This is the GitHub organization for **Equinox**, a Minecraft: Bedrock Edition utility mod for anarchy PvP.

We release the source code for some of our projects and parts of Equinox here when we think our work would benefit the greater native modding community.

The main Equinox project is hosted here as well, but it is closed-source, as the mod is invite-only. **We have no plans to release the source code for Equinox.**
